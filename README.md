Immutable Banking System 

Description
A simple banking system implemented using functional programming concepts like immutability, pure functions, and recursion.

Features
- Create account
- Deposit
- Withdraw
- Transfer money
- View accounts

Concepts Used
- Functional Programming
- Immutability
- Higher-order functions
- Pattern matching
- Maybe type

How to Run
1. Install GHC
2. Run:
   ghci Main.hs
